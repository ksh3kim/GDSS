# Gunpla Finder 🔍

건프라 전 제품을 정밀 분류하여 당신에게 맞는 제품을 추천하는 웹사이트입니다.

## 주요 기능

- **23개 분류 카테고리**: 등급, 시리즈, 난이도, 가동성, 색분할 등
- **다국어 지원**: 한국어 / 영어
- **즐겨찾기 & 비교**: 제품 비교 및 즐겨찾기 기능
- **반응형 디자인**: 모바일/태블릿/데스크톱 지원

## 기술 스택

- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Data**: JSON (정적 데이터)
- **Fonts**: Noto Sans KR, Orbitron
